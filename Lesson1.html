Grammar Explanation

1. N1 は N2 です — Used for identification.
Example: わたし は ダンテ イバラ です。 → I am Dante Ibarra.

2. N1 は N2 じゃありません — Negative form.
Example: ラモスさん は がくせい じゃ ありません。 → Ms. Ramos is not a student.

3. 〜か — Question particle.
Example: イバラさん は フィリピンじん です か。 → Is Mr. Ibarra Filipino?

4. 〜も — “Also” or “too”.
Example: タナビさん も エンジニア です。 → Mr. Tanabi is also an engineer.

5. N1 の N2 — Possession or connection.
Example: イバラさん は WU-P の しゃいん です。 → Mr. Ibarra is a WU-P employee.

6. 〜さん — Polite suffix.
Example: イバラさん → Mr. Ibarra
